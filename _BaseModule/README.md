# Thailand Vanilla Theme Overrider 

Hello and สวัสดี. This is a Vanilla Theme Overrider that turns Vanilla Signs, Road Markings and Decals to Thailand ones.
**Please use one Vanilla Overrider at one time!**

---

## Functions
- Works **automatically**, you don't need any knowledge whatsoever!
- Variable working Street Signs
- Variable working Bus/Tram Signs with max. 5 Lines displayed
- Bicycle Parking Spots


---
